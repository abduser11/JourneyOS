# Prompts — AI-Assisted Development Templates

This directory stores reusable prompt templates for AI coding assistants and code generation tools. Each subdirectory contains prompts optimized for a specific tool, ensuring consistent output quality regardless of which assistant is used.

## Subdirectories

| Folder | Purpose |
|---|---|
| `manus/` | Prompts designed for Manus agent workflows (scaffolding, code generation, refactoring) |
| `claude/` | Prompts optimized for Claude / Anthropic's AI assistant |
| `cursor/` | Prompts for Cursor IDE's AI pair programmer |
| `v0/` | Prompts for Vercel v0's component and UI generation |

## Usage

Each prompt file should include:
1. A clear description of the task
2. Required context (project structure, conventions, constraints)
3. Expected output format
4. Examples where applicable

Prompts are version-controlled so the entire team benefits from iterative improvement.
