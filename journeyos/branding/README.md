# Branding — Visual Identity Assets

This directory is the single source of truth for all JourneyOS brand assets. Every logo, icon, mockup, and marketing visual lives here — versioned alongside the codebase so designers and developers always reference the same canonical files.

## Subdirectories

| Folder | Purpose |
|---|---|
| `logos/` | Primary logo, wordmark, monogram, and dark/light variants (SVG + PNG) |
| `icons/` | Custom brand icons not covered by Lucide (favicon, app icon, opengraph) |
| `mockups/` | Product screenshots, device frames, and marketing visuals |
| `app-store/` | App Store and Play Store listing assets (screenshots, feature graphics) |
| `social-media/` | Platform-specific social media assets (Twitter/X, LinkedIn, Instagram) |

## Conventions

- All vector assets use SVG. Raster fallbacks use PNG at 2x resolution.
- File names are kebab-case: `logo-primary-dark.svg`, `app-icon-512.png`.
- Brand colors and typography guidelines are documented in `docs/02-Design-System.md`.
